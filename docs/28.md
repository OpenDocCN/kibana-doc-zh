# Discover（发现）

您可以从 **Discover**（发现）页面以交互的方式来探索数据。您可以访问每个索引中与所选索引模式匹配的每个文档。您可以提交搜索查询，过滤搜索结果以及查看文档数据。您还可以查看与搜索查询匹配的文档数，并获取字段值统计信息。如果为所选索引模式配置了时间字段，文档随时间的分布被显示在页面顶部的直方图中。
