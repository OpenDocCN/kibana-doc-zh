控制台带有一套实用的键盘快捷键，使得它的工作更高效。这里是一个概述 : 

### 普通编辑

Ctrl/Cmd + I

 自动缩进当前请求。

Ctrl + Space

打开自动完成（即使不输入）。

Ctrl/Cmd + Enter

 提交请求。

Ctrl/Cmd + Up/Down

跳转到 上一个/下一个 请求，开始或结束。

Ctrl/Cmd + Alt + L

折叠/展开当前选择的范围。

Ctrl/Cmd + Option + 0

折叠除当前以外的所有范围。通过添加一个 **shift** 来展开。

### 在自动完成时可见

Down arrow（向下箭头）

将焦点切换到自动完成菜单。使用箭头进一步选择一个词语。

Enter/Tab（回车键/制表键）

选择当前选择的或自动完成菜单中最顶部的词语。

Esc（退出）

关闭自动完成菜单。
