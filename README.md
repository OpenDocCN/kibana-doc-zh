# Kibana 5.2 中文文档

+   [在线阅读](https://kibana.flygon.net)
## 下载

### Docker

```
docker pull apachecn0/kibana-doc-zh
docker run -tid -p <port>:80 apachecn0/kibana-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g kibana-doc-zh
kibana-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
